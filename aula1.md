## Primeira aula

O compilado tem como input um código fonte, output um código alvo ou talvez uma mensagem de erro.

A construção consiste em duas fases:

1. Fase de análise
    - Análise linear
    - Análise hierárquica
    - Análise semântica

2. Fase de síntese
    - Geração de código intermediário
    - Otimização de geração de código;
